# Play Store Assets Brief

This guide gives you a clean first-pass asset plan for `Infinite Wordle`.

## Visual Direction

- Use the warm cream and deep charcoal palette already present in the app.
- Focus on the board, keyboard, and result states instead of device mockup clutter.
- Keep text short and bold so it stays readable on small phones.
- Show gameplay, streaks, and hard mode clearly.

## Required Core Assets

- App icon: 512 x 512 PNG
- Feature graphic: 1024 x 500 PNG
- Phone screenshots: portrait, at least 2

## Feature Graphic Layout Suggestion

- Left side: headline and subhead
- Right side: enlarged tile row and keyboard hints
- Background: soft gradient from cream to pale blue
- Accent: use green and gold tile colors for recognizable gameplay cues

## Screenshot Design Notes

- Keep one main message per screenshot
- Avoid tiny body text
- Use consistent top padding and title placement across all screenshots
- Do not mention anything the app does not currently support
- Do not claim rankings like `best` or `#1`

## Fast Capture Plan

1. Run the Android app in a release-like build.
2. Prepare one device save state for each screenshot.
3. Capture raw screenshots without text overlays first.
4. Add headlines in Figma, Canva, or Photoshop.
5. Export final PNGs at Play-safe sizes.

## Good Screenshot Order For First Upload

1. Endless gameplay
2. Stats and streaks
3. Hard mode
4. Save and resume
5. Clean offline play
